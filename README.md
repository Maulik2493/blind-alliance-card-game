# Blind Alliance 🃏

A trick-taking card game for 3–10 players with hidden team formation, bidding, and trump mechanics.

## Overview

**Blind Alliance** is a points-based trick-taking game where one player wins the bid,
declares a trump suit, and secretly designates teammates using card-reveal conditions.
Teammates are exposed gradually during play — nobody knows who's allied with whom until
the cards give them away.

The game is named after its central tension: you're in an alliance, but everyone at the
table is blind to who's truly on whose side.

> Originally designed as "Spades of 3", named after its highest-value card:
> the **3 of Spades** (worth 30 points).

## Quick Facts

| Property        | Value                                      |
|-----------------|--------------------------------------------|
| Players         | 3–10                                       |
| Decks           | 1 deck (3–5 players), 2 decks (6–10)       |
| Total Points    | 250 per deck                               |
| Min Bid         | 125 (1 deck) / 250 (2 decks)              |
| Win Condition   | Bidder's team scores ≥ bid                 |
